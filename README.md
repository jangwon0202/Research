
**수학기호는 좌우에 $붙이기**

1.거듭제곱 (^)  ----- (ex) (`2^N`)
-$2^N$  $2^{20}$ 
2.로그 (\log) ---- (ex) (`\log N`, \log_2 N)
-$log_2 N$ $\log N$ 
3.분수 ( \frac{분자}{분모} ) - (ex) (`\frac{N(N+1)}{2}`)
-$\frac{10}{3}$ $\frac{N(N+1)-3}{(N-2)+4}$  
4.곱셈 기호 (\times):  `*` 대신 `\times` --- (ex) $N \times M$)
-$(11\times3)$  $N\times(N+2)$ 
5.조합 (_nC_r) - (ex)  ('_5C_3') 
-$_7C_4$ 
6.시그마 (\sum_{i=1}^{n} i)
-$\sum_{i=1}^{n} i$

7.부등호 
$\le$ (\le) - 작거나 같다
$\ge$ (\ge) - 크거나 같다
$\neq$ (\neq) - 같지 않다
$\approx$ (\approx) - 거의 근사함
$\equiv$ (\equiv) - 합동 혹은 정의됨
