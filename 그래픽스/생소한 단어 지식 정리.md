


**셰이딩(Shading)** - 빛과 그림자, 색을 계산해서 입히는 모든 과정

**컬링(Culling)** - 화면에 안 보이는 쓸데없는 걸 골라내서 버리는 작업.

**클러스터(Cluster)** - 연관 있는 것들끼리 묶어놓은 덩어리


**비저블 클러스터** - 화면에 실제로 보일 가능성이 있는 3차원 공간이나 기하학(Geometry)의 덩어리(Cluster)

**클러스터드 셰이딩(Clustered Shading)** - 게임 화면을 3차원 상자 덩어리(클러스터)들로 나누고 각 구역(덩어리)마다 필요한 조명만 골라서 칠해지는 기술

**메시 컬링(Mesh Culling)** - 그물망(Mesh)처럼 얽혀 있는 복잡한 3D 오브젝트 중에서 카메라에 안 보이는 부분은 과감하게 솎아내서(Culling) 렌더링 계산에서 제외하는 기술

**프러스텀 컬링(Frustum Culling)** - 카메라의 시야 범위(절두체 모양) 밖에 있는 오브젝트들은 그래픽 카드가 아예 계산을 안하도록 하는 기술

**오클루전 컬링(Occlusion Culling)** - 시야 범위 안에 있더라도, 큰 건물이나 벽에 가려져서 '실제로 눈에 안 보이는 뒤쪽 사물'을 판별해서 그리기를 생략하는 기술

**LOD(Level of Detail)** - 카메라와 가까운 사물은 정밀하게(ex:수만 개의 삼각형), 멀리 있는 사물은 대충(ex:수십 개의 삼각형) 그리도록 실시간으로 메시를 교체하는 기술

**메시(Mesh)** - 입체적인 물체(Object)를 수많은 점과 선, 다각형 등으로 구성한 그물망 형태의 3D 모델 데이터

**SSR(Screen Space Reflection)** - 화면에 찍힌 정보만을 활용해 도로 위의 물웅덩이나 차량 표면에 주변 환경을 실시간으로 반사하는 기법

**Compute Shader 기반 센서 모방** - 가상의 레이저를 수만 발 쏘는 라이다(LiDAR)나 초음파 센서의 물리적 충돌 연산을 GPU의 병렬 연산(Compute Shader)으로 처리하여 데이터화하는 기술.

**깊이/열화상 버퍼 렌더링** - 화면을 일반 색상이 아닌, 카메라와의 거리를 기반으로 한 깊이 값(Depth)이나 적외선 센서용 흑백/열화상 텍스처르 뽑아내는 멀티 렌더 타겟(MRT) 기술

**카메라 매칭 (Matrix Synchronization):** 실제 촬영 현장의 카메라 위치/초점 거리 데이터를 실시간으로 받아와, 언리얼 엔진 속 가상 카메라의 View Matrix와 Projection Matrix에 1ms의 오차도 없이 실시간 동기화하는 기술.

**Genlock (제너레이터 락):** 실제 카메라의 셔터 타이밍과 LED 스크린의 주사율(프레임)을 완벽하게 일치시켜, 화면이 지직거리거나 검은 줄(롤링 셔터 현상)이 가는 걸 막는 하드웨어 연동 기술.

**ACES (Academy Color Encoding System):** 영화 산업 표준 색공간. 모니터, LED, 카메라 기기마다 제각각인 색 표현을 수학적으로 통일하는 컬러 파이프라인 구축 기술.

**레이 트레이싱 (Ray Tracing):** 빛의 알갱이가 튕겨 나가는 경로를 역추적하여, 유리창의 굴절과 정밀한 그림자를 물리적으로 완벽하게 시뮬레이션하는 고급 렌더링 기술.

**볼륨 렌더링 (Volume Rendering)**

- **레이 마칭 (Ray Marching / Ray Casting):** 일반적인 3D 캐릭터는 속이 텅 빈 삼각형 껍데기지만, CT 데이터는 속이 꽉 차 있어. 카메라에서 빛을 발사해 이 3D 데이터 볼륨 내부를 통과시키면서, 밀도에 따라 색을 누적해 정밀한 내부 장기를 표현하는 기술.
  
- **등가면 추출 (Isosurface Extraction / Marching Cubes):** 3D 밀도 데이터에서 "뼈의 밀도에 해당하는 부분만 골라내서 삼각형 메쉬 모양 껍데기로 자동 변환해라" 할 때 쓰는 알고리즘.
