

**(1)input.GetAxis()**
키보드나 마우스, 조이스틱을 얼마나 움직였는지 알려주는 함수

input.GetAxis(InputAxis::XAxis)는 가로축(좌우 움직임)
input.GetAxis(InputAxis::YAxis)는 세로축(위아래 움직임)
input.GetAxis(InputAxis::ZAxis)는 깊이축(앞뒤 움직임)

**(2) .GetNormalize()**
정규화 함수

가로가 1 세로가 1일 때 대각선을 $\sqrt{2}$가 아닌 1로 맞춰줌.
가로가 5 세로가 5일 때는 대각선을 5로 맞춤.

**(3) Math::GetSinCos()**
Math::GetSinCos(sin,cos,Degree)


**(4)`_ScreenSize.X, _ScreenSize.Y`**

`_ScreenSize.X, _ScreenSize.Y`는 현재 모니터의 해상도를 나타냄.
FHD(1920X1080)일때 ScreenSize.X는 1920, ScreenSize.Y는 1080

**(5) HSVColor hsv()**
HSVColor hsv(0.f, 1.f, 0.85f)
- H(Hue, 색상) : 0.0 ~ 1.0($0 \sim 360^\circ$) 사이의 값.
- S(Saturation, 채도) : 색의 선명도. 1.f는 진한 원색을 쓰겠다는 뜻.
- V(Value, 명도) : 색의 밝기. 0.85f는 살잒 차분하고 밝은 톤
  
HSV 형태를 사용했기에 모니터가 이해할 수 있도록 hsv.ToLinearColor()로 RGB형태로 변환해서 넘겨줌.

**(6) Math::Lerp(A, B, t)**
Math::Lerp(1.f, 5.f, ratio)

선형 보간 함수, 시작점(A)와 끝점(B)사이를 비율(t)만큼 비례하여 나눈 위치의 값을 반환함
공식은 `A + (B - A) X t`
t가 0.0일 때 A 반환
t가 1.0일 때 B 반환
0.0< t <1.0일 때 사이 값 반환

**(8)ToCartesianCoordinate()**
데카르트(직교) 좌표계로 변경하는 함수. (Cartesian = 데카르트)

**(9)Math:Deg2Rad()**

인간의 각도(Degree)를 컴퓨터의 각도(Radian)의 각도로 변환해주는 변환 상수이다.
값은 ($\frac{\pi}{180}$)
