## 📌 0단계: 선수 지식 및 기초 다지기 (Prerequisites)

> 본격적인 그래픽스를 시작하기 전, 데이터를 다루기 위한 기본 체력을 기르는 단계

- [ ] **C++ 심화 (Modern C++)**
    
    - 포인터와 메모리 동적 할당 (`new`, `delete`), 스마트 포인터 이해
        
    - 구조체(struct)와 클래스(class)의 메모리 레이아웃 및 정렬(Alignment)
        
    - 가상 함수(Virtual Function)와 vtable의 동작 원리
        
- [ ] **그래픽스 수학 (선형대수학)**
    
    - 벡터(Vector): 내적(Dot Product)과 외적(Cross Product)의 기하학적 의미 및 실무 응용
        
    - 행렬(Matrix): 회전, 크기, 이동 행렬의 결합 연산 공식
        
    - 좌표계 변환: 로컬(Local) $\rightarrow$ 월드(World) $\rightarrow$ 뷰(View) $\rightarrow$ 투영(Projection) 공간으로의 수학적 흐름 마스터
        

## 🎨 1단계: 테크니컬 아트 및 셰이더 기초 (Shader Tech)

> 현재 네가 맛보기 시작한 단계야. GPU가 일하는 논리식을 시각적으로 이해하는 구간이지.

- [ ] **노드 기반 머티리얼 시스템 마스터**
    
    - 채널 패킹 이미지 활용 기법 및 VRAM 최적화 이론 정리
        
    - 마스터 머티리얼 구조 설계 (Panner, Math, Masking, Output 모듈화)
        
    - PBR(물리 기반 렌더링) 입력 핀 마스터 (Albedo, Metallic, Roughness, AO, Normal, Subsurface)
        
- [ ] **고급 셰이더 기법 구현해보기**
    
    - 프레넬(Fresnel) 연산을 활용한 SF 홀로그램 재질 만들기
        
    - 두께 지도(Thickness Map)와 SSS를 활용한 반투명 피부/물 렌더링 구현
        
    - 월드 포지션 오프셋(WPO)을 이용한 펄럭이는 깃발, 일렁이는 불꽃 구체 연출
        

## 🛠️ 2단계: 렌더링 파이프라인과 HLSL 코드 셰이더 (Mid Level)

> 마우스 노드 연결을 끊고, 실제 그래픽스 코드를 타이핑하며 GPU 내부 컨베이어 벨트를 통제하는 단계야.

- [ ] **GPU 고정 기능 파이프라인 (Graphics Pipeline) 이해**
    
    - 정점 버퍼(Vertex Buffer)와 인덱스 버퍼(Index Buffer)의 구조
        
    - 래스터라이제이션(Rasterization) 단계에서 3D 정점이 2D 픽셀로 변환되는 과정
        
- [ ] **HLSL (High-Level Shader Language) 프로그래밍**
    
    - 유니티 셰이더 그래프나 언리얼 커스텀 노드에서 텍스트 코드로 셰이더 작성하기
        
    - **정점 셰이더 (Vertex Shader):** 정점 위치를 직접 비틀어 파도나 바람 효과 구현
        
    - **픽셀 셰이더 (Pixel Shader):** 픽셀 단위 텍스처 샘플링 연산 및 조명(Lighting) 연산 코딩
        
- [ ] **수학적 조명 모델(Lighting Model) 직접 구현**
    
    - 램버트(Lambert) 디퓨즈 연산 공식을 코드로 구현하기 (`dot(Normal, LightDir)`)
        
    - 퐁(Phong) 및 블린-퐁(Blinn-Phong) 스페큘러 반사광 공식 연산
        

## 🌋 3단계: 로우 레벨 그래픽스 API 프로그래밍 (High Level)

> 게임 엔진의 보호막을 걷어내고, C++ 코드로 그래픽 카드 하드웨어를 직접 컨트롤하는 최종 종착지야.

- [ ] **DirectX 11 또는 DirectX 12 시작하기 (윈도우 환경 기준)**
    
    - Win32 API로 기본 하얀색 창 띄우기
        
    - GPU 장치 초기화 (`CreateDevice`, `SwapChain` 설정)
        
- [ ] **C++ 코드로 3D 삼각형 띄우기 (Hello Triangle)**
    
    - RAM에 있는 정점 배열 데이터를 VRAM(비디오 메모리)으로 전송하는 파이프라인 구축
        
    - HLSL 소스코드를 로드해서 GPU에 컴파일 및 바인딩하기
        
- [ ] **자체 미니 렌더러 제작 (포트폴리오 코어)**
    
    - `.obj`나 `.fbx` 모델 파일을 C++ 파서로 읽어와 화면에 렌더링하기
        
    - 카메라도 마우스 제어 코드(FPS 카메라)를 C++로 직접 계산해서 행렬에 업데이트하기
